## 핵심 기능
#### 토큰  사용
- JWT토큰을 이용한 클라이언트, 서버간 인증

#### bcrypt를 이용한 암호화
- hashlib의 보안 취약점을 보안하기 위한 사용

#### 명함의 편리한 검색, 즐겨찾기 구현
- 명함에서 검색 및 즐겨찾기 등의 정렬을 구현해 사용자가 쉽게 명함을 파악
